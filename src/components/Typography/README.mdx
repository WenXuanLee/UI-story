# Typography

The Typography component provides a consistent and flexible way to manage text styles across your UI library. It supports different **types** (e.g., headings, body text, captions), **states** (e.g., error, success), and **weights** for various use cases and design consistency.

## **Best Practices**

1. **Use `h1` only for primary page titles**. Avoid skipping heading levels for SEO.
2. Use **dynamic states** like `error` or `success` for feedback messages.
3. Keep **weights** (e.g., `semiBold`, `bold`) consistent for headings and subheadings.
