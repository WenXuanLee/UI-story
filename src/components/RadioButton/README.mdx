# Radio Button

The **Radio Button** component allows users to select **one option from a set**. Itâ€™s commonly used in forms where a single selection is required, such as choosing a **payment method**, **subscription plan**, or **filtering options**.

---

## **Best Practices**
1. **Use Radio Buttons when only one option can be selected** from a group.
2. **Keep labels short and clear** so users can easily understand the choices.
3. **Ensure the selected option is clearly visible** with contrast against unselected states.
4. **Avoid using Radio Buttons for binary choices**; use a **Toggle (Switch)** or **Checkbox** instead.

## **Usage Example: Radio Group**
For most use cases, **Radio Buttons** should be used inside a `RadioGroup` component.

<Story name="RadioGroupExample" />

### **Code Example**
```tsx
import { RadioGroup } from "./RadioGroup";

export const ExampleUsage = () => {
  return (
    <RadioGroup
      name="subscription"
      options={[
        { label: "Basic Plan", value: "basic" },
        { label: "Standard Plan", value: "standard" },
        { label: "Premium Plan", value: "premium" },
      ]}
      defaultValue="standard"
      size="md"
      state="default"
    />
  );
};
