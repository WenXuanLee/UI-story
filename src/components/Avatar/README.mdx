# Avatar

The **Avatar component** is used to display a userâ€™s profile image, initials, or a fallback icon. It supports multiple sizes, shapes, and an optional status indicator to represent online/offline status.

---

## **Best Practices**

1. **Use images whenever possible** to provide a clear user identity.  
2. **Fallback to initials or an icon** for users without a profile image.  
3. **Status indicators** are useful for representing user presence.

## **Avatar Sizes and Shapes**

<Story name="WithInitials" />
