# Toast

The **Toast component** provides feedback messages like **success, warning, error**, and **informational notifications**. It supports different **positions**, can be **dismissed manually**, or **auto-dismissed** after a duration.

---

## **Best Practices**
✅ Keep toast messages **short and concise**.  
✅ Use **`autoDismiss`** for temporary status updates.  
✅ Use `variant` props to **indicate the type of feedback**.
