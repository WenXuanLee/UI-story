# Tooltip

The **Tooltip component** is used to display small contextual information when users hover over an element. It supports **dynamic positioning** and automatically adjusts based on content length.

---

## **Best Practices**
✅ Keep tooltips **short and concise** to avoid overwhelming users.  
✅ Use tooltips for **additional hints**, not for essential information.  
✅ Ensure tooltips are **close to the element they describe**.  
✅ Set a **`maxWidth`** for tooltips to ensure proper readability.  
✅ **Long words will wrap automatically** to prevent overflow issues.
