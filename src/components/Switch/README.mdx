# Switch (Toggle)

The **Switch component** allows users to toggle between two states, typically **ON** and **OFF**.

---

## **Best Practices**
1. **Use Switches for binary state changes** (e.g., Enable/Disable settings).
2. **Avoid using switches for actions** that require confirmation.
3. **Use labels** to provide context for what the toggle controls.
4. **Ensure accessibility** so users can toggle via **keyboard (Space / Enter)**.
5. **Use a disabled state** when the switch should not be interacted with.
