# Chips

The **Chips** component is a compact, interactive UI element used for filtering, tagging, or indicating selections.  
It supports **different styles, sizes, icons, and removable options**.

---

## **Best Practices**
1. **Use Chips to represent selections or filters.** Avoid using them for critical actions.
2. **Ensure sufficient padding and spacing** between multiple chips.
3. **Clickable chips should have visual feedback** when hovered or focused.
4. **Removable chips should always have a clear close (`✖`) button.**
5. **Use outlined variants for a less prominent look** when appropriate.

## **Usage Example: Clickable Chips**
Clicking on a Chip can trigger an action, such as filtering a list.

<Story name="Clickable" />

### **Code Example**
```tsx
<Chip label="Clickable Chip" variant="primary" onClick={() => alert("Chip clicked!")} />
```

---

## **Removable Chips**
Chips can be removable by adding the `removable` prop.

<Story name="ClickableRemovable" />

### **Code Example**
```tsx
<Chip label="Removable Chip" variant="primary" removable onRemove={() => alert("Chip removed!")} />
```

## **Conclusion**
The **Chips component** is useful for tagging, filtering, and selection indicators.  
For best results:
- **Use outlined variants for subtle UI elements**.
- **Ensure removable chips have an `✖` button** for easy deletion.
- **Make sure chips have enough spacing when used in groups.**
