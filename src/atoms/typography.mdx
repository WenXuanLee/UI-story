# Typography Guide Line

**Weights:** different size will have limited weight options
- 400 (regular): body, caption, small
- 500(medium): subheading, small-subheading
- 600(semi-bold): all sizes
- 700(bold): all sizes
- 800(extra-bold): h1, h2, h4

**Size:** Please enusre what kind of text styles applied on the design figma.
- 28px: h2 
- 22px: h2 
- 18px: h3 
- 16px: h4, subheading, body
- 14px: small-subheading, caption
- 12px: small
